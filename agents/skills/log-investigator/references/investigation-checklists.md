# Investigation Checklists

## 7项调查检查清单（卡住时）

- [ ] 逐字读完错误日志/堆栈？
- [ ] 搜索过核心关键词？
- [ ] 读过源码上下文 50 行？
- [ ] 所有假设都确认过？
- [ ] 试过相反假设？
- [ ] 能最小范围复现？
- [ ] 换过查询角度/时间范围？

## 常见借口

| 借口 | 反击 | 行动 |
|------|------|------|
| "我知道，直接修" | 每次都值得新调查 | 从 Step 1 开始 |
| "跳过假设表" | "明显"修复导致级联故障 | 强制填表 |
| "不用查 knowledge" | 历史问题省80%调查时间 | 必须先查 |
| "可能是环境问题" | 验证过吗？ | 用工具确认 |
